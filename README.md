# vezdecode_cpp30
Решение задания Вездекод C++30
# Install and run
```
sudo apt install libcurl4-openssl-dev
mkdir build
cd build
cmake ..
cmake --build .
./vezde30
```

# Дополнительное задание
Чтобы ввести название метода, достаточно написать:
```
./vezde30 <method_name>
```
Например:
```
./vezde30 vezdecode30_req
```
